## 一、设计范式

| 关键词 | 说明 |
|--------|------|
| **编辑设计 (Editorial Design)** | 参考 area17.com，杂志级排版气质，图文交错叙事 |
| **Apple-esque 极简** | 大量留白、克制的元素、精致的微交互、噪点胶片质感 |
| **高级商务** | 全景落地窗办公背景 + 城市天际线 + 浮动光球氛围 |
| **橄榄绿单色调** | 仅用一种强调色（#5C6B3A），所有高亮、hover、图标统一 |

## 二、色彩系统（完整 CSS 变量）

```css
:root {
  --bg:        #FAFAF8;   /* 微暖白，页面主背景 */
  --bg-warm:   #F5F4F0;   /* 暖灰，section 背景 */
  --surface:   #FFFFFF;   /* 纯白，卡片背景 */
  --text:      #1A1A1A;   /* 近黑，主标题 */
  --text-secondary: #5A5A5A;  /* 中灰，正文描述 */
  --text-muted:     #999999;  /* 浅灰，辅助信息 */
  --accent:    #5C6B3A;   /* 橄榄绿，唯一强调色 */
  --accent-hover:   #4A5A2E;  /* 深橄榄绿，hover 态 */
  --accent-soft:    rgba(92, 107, 58, 0.08);  /* 极淡绿，背景 tint */
  --border:    rgba(0, 0, 0, 0.06);   /* 极淡边框 */
  --border-hover:   rgba(0, 0, 0, 0.12);   /* hover 边框 */
  --shadow-sm: 0 1px 3px rgba(0,0,0,0.03);
  --shadow-md: 0 8px 32px rgba(0,0,0,0.06);
  --shadow-lg: 0 24px 60px rgba(0,0,0,0.08);
  --radius-sm: 8px;
  --radius-md: 14px;
  --radius-lg: 20px;
  --transition: cubic-bezier(0.4, 0, 0.2, 1);
}
```

### 色彩使用规则

- **强调色唯一性**：所有强调、hover、图标、编号、标签高亮都用 `--accent`
- **层级构建**：`--bg`（页面底）→ `--bg-warm`（section 底）→ `--surface`（卡片）
- **边框克制**：默认几乎不可见，hover 时才轻微显现
- **文字三层**：`--text`（标题）→ `--text-secondary`（正文）→ `--text-muted`（辅助）

## 三、字体系统

### 字体栈

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600&family=Playfair+Display:ital,wght@0,400;0,500;0,600;0,700;1,400;1,500&family=JetBrains+Mono:wght@400;500&display=swap');
```

| 用途 | 字体 | 字重 | 字间距 | 行高 |
|------|------|------|--------|------|
| **Display 大标题** | Playfair Display | 400 | -0.035em | 1.1 |
| **Section 标题** | Playfair Display | 400 | -0.03em | 1.15 |
| **斜体强调** | Playfair Display Italic | 400 | -0.02em | 1.15 |
| **正文 / UI** | Inter | 400/500 | -0.01em | 1.6 |
| **标签 / 编号** | JetBrains Mono | 400/500 | 0.02~0.2em | 1 |

### 字号层级

| 元素 | 字号 | 说明 |
|------|------|------|
| Hero 标题 | clamp(2.5rem, 6vw, 4.5rem) | 最大字号，视觉冲击 |
| Section 标题 | clamp(2rem, 5vw, 3.5rem) | 模块分隔 |
| 卡片标题 | 1.1~1.2rem | 粗体 600 |
| 正文 | 0.8~0.85rem | 颜色 --text-secondary |
| 标签/Tag | 0.65~0.75rem | Mono 或 Inter 500 |
| 辅助信息 | 0.65~0.7rem | 颜色 --text-muted |

## 四、布局规则

### 容器

```css
.container {
  max-width: 1280px;
  margin: 0 auto;
  padding: 8rem 4vw;       /* 大留白 */
  position: relative;
  z-index: 1;
}
```

- **Hero 特殊边距**：`padding: 8rem 8vw 4rem 12vw`（左侧更宽，内容居中）
- **Section 圆角包裹**：部分 section 使用 `margin: 0 2vw` + `border-radius: var(--radius-lg)` 形成内嵌卡片感

### 网格系统

| 场景 | 列数 | 间距 |
|------|------|------|
| 三列卡片 | `repeat(3, 1fr)` | 2~3rem |
| 两列卡片 | `repeat(2, 1fr)` | 2~3rem |
| 图文交错 | `1fr 1fr` | 3rem |
| 经历时间线 | `280px 1fr`（左信息右内容）| 3rem |

### 圆角与阴影

```css
/* Pill / Tag */
border-radius: 100px;

/* 卡片 */
border-radius: var(--radius-md);  /* 14px */
background: var(--surface);
border: 1px solid var(--border);

/* 大卡片 / Section */
border-radius: var(--radius-lg);  /* 20px */

/* 阴影 */
box-shadow: var(--shadow-sm);  /* 默认 */
hover: box-shadow: var(--shadow-md);  /* hover */
```

## 五、组件规范

### 5.1 导航（Mega Menu）

- 中英文双标签：`首页 <span class="font-mono">Home</span>`
- hover 下拉白色面板：`background: var(--surface)` + `box-shadow: var(--shadow-lg)` + 小三角
- 当前项下划线：`height: 1.5px; background: var(--accent); transform: scaleX(0→1)`
- 联系按钮：线性风格（透明底 + 1.5px 橄榄绿边框），hover 填充


```css
.card {
  background: var(--surface);
  border: 1px solid var(--border);
```

- 字体：Playfair Display，2~3rem
- 默认 opacity: 0.2，hover 提升至 0.4 并可变色为 `--accent`

## 六、交互动效

### 6.1 滚动渐入（Scroll Reveal）

```css
.reveal {
  opacity: 0;
  transform: translateY(40px);
  transition: opacity 0.8s var(--transition), transform 0.8s var(--transition);
}
.reveal.visible {
  opacity: 1;
  transform: translateY(0);
}
.reveal-delay-1 { transition-delay: 0.1s; }
.reveal-delay-2 { transition-delay: 0.2s; }
.reveal-delay-3 { transition-delay: 0.3s; }
```

### 6.2 卡片 Hover

```css
transform: translateY(-4px);
border-color: var(--border-hover);
box-shadow: var(--shadow-md);
transition: all 0.4s var(--transition);
```

### 6.3 导航下划线

```css
/* 从中间向两侧展开 */
transform: scaleX(0);
transition: transform 0.4s var(--transition);
:hover::after { transform: scaleX(1); }
```

### 6.4 浮动光球（页面氛围）

```css
.orb {
  position: fixed;
  border-radius: 50%;
  pointer-events: none;
  z-index: 0;
  filter: blur(80px);
  animation: orbFloat 18s ease-in-out infinite;
}
@keyframes orbFloat {
  0%, 100% { transform: translate(0, 0) scale(1); }
  33% { transform: translate(-40px, 30px) scale(1.05); }
  66% { transform: translate(20px, -20px) scale(0.95); }
}
```

### 6.5 噪点纹理

## 九、【S级品牌活动】详情页建议结构

基于以上设计系统，详情页推荐采用以下结构：

```
1. 导航（复用主站 Mega Menu）
2. 案例 Hero（全宽背景图 + 渐变遮罩 + 大标题 + 关键数据）
3. 项目概述（两栏：左文右图或左图右文）
4. 核心策略（三列卡片，带编号 01/02/03）
5. 执行过程（时间线或步骤卡片）
6. 数据成果（大号数字展示 Playfair Display）
7. 关键素材（大图画廊，圆角 + 阴影）
8. CTA / 返回首页
```

**必须复用的元素**：
- 色彩变量（一字不改）
- 字体栈（Playfair + Inter + JetBrains Mono）
- 圆角和阴影系统
- `--transition: cubic-bezier(0.4, 0, 0.2, 1)`
- 滚动渐入 `.reveal` 类
- 噪点纹理 `.noise-overlay`
- 浮动光球 `.orb`
- Pill 标签样式
